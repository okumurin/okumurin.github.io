6.9 重複テスト分冊法

内容領域の網羅と回答者の負担の軽減

テストの得点をいかに使うかということは、そのテストの用途に依存する。 PISAやTIMSS、NAEPのような大規模な学力調査では、個人の学力の状態を診断することが目的ではなく、受験者集団全体の学力の動向を把握するのが目的である。 こういった調査では、調査全体では多様な項目を提示した方が広い領域からまんべんなく出題できて、妥当性を高く保つことができる。 しかしながら、多数の項目を回答するのは、個々の回答者には大きな負担となる。 また、個人の結果が返されないような学力調査では、そのような負担の大きいテストに答える動機づけは低く、調査結果の妥当性をも低めかねない。

従って、こうした学力調査では、項目全体をいくつかの冊子に分割し、回答者はいずれかの冊子を割り当てられて回答するという形式が取られることが多い。 このようにすることで、調査全体では多くの項目を出題することができ、かつ個々の回答者の負担は軽くすることが可能である。

釣り合い型不完備ブロックデザイン

Table 16 は、PISAで採用されている冊子分割のデザインである。 まず、項目全体を13のブロックに分け、1つの冊子に4つのブロックが含まれるようにし、合計で13の冊子を作る。 例えば、1番目の冊子には、ブロック1、2、4、7がこの順で印刷されている。 回答者はどれか1つの冊子のみに回答する。これら13の冊子は1冊ずつ積み上げられ、それが順番に回答者に配布される。 こうすることで、すべての冊子がまんべんなく回答者に配布される。

Table 16 は、

  1. どのブロックも必ず4回ずつ含まれる

  2. どのブロックも必ず4つの場所いずれか1回ずつで提示される

  3. 任意の2つのブロックの組み合わせが必ず1回だけ存在する

  4. 任意の2つの冊子で共通のブロックが必ず1つ存在する

という特徴を持っている。

回答者ごとに異なる冊子を回答していても、冊子同士は必ず共通のブロックが含まれているため、これを手がかりとして、逐次全ての冊子の等化を行うことができる。 最終的に、どの冊子を回答しても、能力が互いに比較可能となる。

また、受験者集団全体で見ると、どのブロックも同じ回数、前半後半全ての場所で、また他のブロックとの組み合わせもバランスが取られている。 このようなデザインを、釣り合い型不完備ブロックデザイン（balanced incomplete balance design: BIBD）と呼び、BIBDにもとづいて問題冊子を設計することをBIBスパイラル方式などという。 また、複数の冊子にブロックが重複していることから、日本語では重複テスト分冊法と呼ばれることもある。 TIMSSやNAEPでも同様の冊子分割法が取られている。

回答者ごとに異なる冊子を回答することは、負担を軽減しながら調査全体の内容は犠牲にしないとともに、不正行為を防止するためにも有効である。
