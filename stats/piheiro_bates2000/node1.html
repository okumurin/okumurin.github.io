線形混合効果モデル：基本概念と例

lmeによる変量効果モデルのあてはめ

  • update()：前回のあてはめから異なる引数だけ指定してモデルを更新 

> fm1 <- lme(y ~ x, …)
> fm2 <- update(fm1, . ~ . + z) # fm1にzを加える
> fm3 <- update(fm2, . ~ . - z) # fm2からzをなくす（fm1と同じ）

説明変数に因子型変数 f が含まれている場合， 

> fm1 <- lme(y~f-1, ...)
のようにすることで，セル平均をパラメータ化することが可能。

特定の変数をセンタリングしたり，高次の項を含めたりしたい場合， 

> fm1 <- lme(y~I(x-10), …)
> fm1 <- lme(y~x+I(x^2), …)
のように関数 I() を用いるとよい。

例えば，データ Orthodont について， 

> fm6 <- lme(distance ~ I(age-11), Orthodont)
> fm7 <- update(fm6, fixed = distance ~ Sex * I(age - 11))
とするなど。

モデルのあてはまりのチェック

  • plot(fm1)：あてはめたモデル fm1 について，あてはめ値に対する標準化残差をプロット
  • anova(fm1)：あてはめたモデル fm1 について，因子全体の総合的な有意性を判断
  • anova(fm1, fm2)：あてはめたモデル fm1 と fm2 について，尤度比検定を実行

REML推定を用いる場合，尤度比検定やAICやBICの比較は
  1. 同じ固定効果構造をもち
  2. かつその固定効果構造中の全ての因子に同じ対比を使用している
場合に限られる。

  • resid(), fitted()：残差やあてはめ値を取り出す。式中に"."を用いると，あてはめモデルそのもの fm1 と解釈される。
例えば， 
> plot(fm1, form=resid(., type="p")~fitted(.) | Subject, abline=0)
であてはめ値と残差のプロットを描ける1

  • augPred(fm1)：与えられた共変量の範囲での予測値を返す
例えば， 
> plot(augPred(fm1), grid=T)
のようにすれば，元のデータに予測値を重ねてプロットを描くことができる。

データ Orthodont を用いた場合， 

> plot(fm6) # 当てはめ値と残差
> plot(augPred(fm6), layout = c(8,4), between = list(y = c(0, 0.5)), grid = T) # データと当てはめ値
で以下の図を描ける。
